# 第六部分：完整製作流程

## 步驟一：歌曲生成（建議先完成歌曲）

1. 前往 **Suno AI**（suno.com）或 **Udio**（udio.com）
2. 貼上本文件「Suno AI 提示詞（完整版）」
3. 貼上完整歌詞（從 Verse 1 至 Final Chorus，分段標註）
4. 生成 3–5 個版本，根據以下條件評分選出最佳：
   - ✅ 人聲音色溫暖，非兒童聲、非滄桑聲
   - ✅ Chorus 有明顯爆發感
   - ✅ Bridge 段明顯安靜，有反差
   - ✅ Final Chorus 明顯比前段更有力量感
5. 下載 WAV（高品質）格式備用

## 步驟二：影片素材生成

1. **依時序逐段生成**：建議從最重要的段落先做（開場、副歌、結尾）
2. **推薦工具順序**：Runway Gen-3 → Kling AI → Pika 2.0
3. **每段生成步驟**：
   - 貼入段落的「AI 生成提示詞（英文）」
   - 附加本文件「主角 AI 生成統一描述」
   - 點擊生成，等待 30-60 秒
   - 生成 2-3 個版本，選畫質最佳且符合情緒的
4. **主角一致性處理**：若不同段落主角外型不一致，可用 Pika 的 IP Reference 或 Runway 的角色鎖定功能

## 步驟三：影片剪輯整合

1. **工具推薦**：
   - 剪映（最簡易，中文介面，適合初學者）
   - CapCut（功能多，有 AI 字幕）
   - DaVinci Resolve（專業，色彩校正最強）

2. **剪輯順序**：
   - 先排好所有素材（依時序表）
   - 加入歌曲音軌（WAV 格式）
   - 根據音樂節奏調整各素材長度
   - 加入轉場效果（依各段落說明）

3. **字幕製作**：
   - 手動或 CapCut AI 字幕功能
   - 統一使用思源黑體字型
   - 位置、大小依字幕設計規範

4. **色調調整**：
   - 套用本文件「整體色調後製設定」的數值
   - 統一所有素材的白平衡和亮度

5. **音效混音**：
   - 歌曲底層音量約 70%
   - 旁白時降至 30%
   - 環境音效始終在 15-40% 範圍

## 步驟四：最終輸出規格

| 用途 | 規格 |
|------|------|
| 主版（3 分鐘） | 1920×1080 / MP4 H.264 / 音訊 AAC 320kbps |

# 第十三部分：剪映影片剪輯逐步教程

> 適合初次使用剪映的使用者。以下為完整操作流程，
> 以手機版剪映（CapCut）為主，部分步驟同樣適用桌面版。

---

## 13-1 前置準備

```
開始剪輯前，請先整理好以下素材：

資料夾結構（建議）：
📁 競選影片製作
  📁 01_影片素材       ← 所有 AI 生成的影片片段（MP4）
  📁 02_音樂          ← 主題曲 WAV 檔、備用曲 WAV 檔
  📁 03_音效          ← 各段落音效 MP3（依音效清單）
  📁 04_照片          ← 主角照片、場景照片
  📁 05_字型          ← 思源黑體 Bold/Heavy（若桌面版需要）
  📁 06_輸出          ← 最終匯出影片存放處

命名建議：
  01_corridor_morning.mp4
  02_rehab_hands.mp4
  03_smile_hallway.mp4
  ...依段落編號命名，方便排序
```

---

## 13-2 匯入素材與排列

```
步驟 1：開啟剪映，點「新建項目」
步驟 2：點「+」匯入所有影片素材（可多選）
步驟 3：依時序拖入時間軌道
  → 按照 01、02、03 順序拖入
  → 總長度目標：180 秒（3 分鐘）

步驟 4：初步對齊時序表
  → 開場（01）：0:00–0:20
  → 復健（02-04）：0:20–0:48
  → 笑臉（05-07）：0:48–1:10
  （以此類推）

步驟 5：預覽整體，確認素材長度足夠
  → 若某段太短，可在剪映中「慢速」延長
  → 建議慢速比例：0.6x–0.8x（不要低於 0.5x）
```

---

## 13-3 加入音樂軌道

```
步驟 1：點下方「音頻」→「本地音樂」
步驟 2：匯入主題曲 WAV 檔
步驟 3：拖入音軌，對齊影片開頭（0:00）
步驟 4：音量設定 → 70%（背景音樂預設值）

步驟 5：加入音效軌道（依音效清單）
  → 每個音效單獨匯入
  → 拖至對應時間點
  → 各音效音量依清單設定

步驟 6：旁白時降低音樂音量
  → 選音樂軌道 → 「音量關鍵幀」
  → 在旁白開始前 0.5 秒：70% → 30%
  → 在旁白結束後 0.5 秒：30% → 70%
```

---

## 13-4 各段落轉場效果設定

| 段落 | 轉場效果 | 剪映操作位置 |
|------|---------|------------|
| 開場 → 復健 | J-Cut（音先畫後） | 在素材間拖重疊 0.5 秒 |
| 復健 → 笑臉 | Whip Pan 甩鏡 | 「轉場」→「運鏡」→「甩鏡」 |
| 笑臉 → 超人 | 慢動作最後一幀硬切 | 在最後一幀加「定格」0.3 秒後硬切 |
| 超人 → 宣言 | 硬切（Smash Cut） | 直接切，不加轉場 |
| 宣言 → 情誼 | 光暈轉場 | 「轉場」→「光效」→「鏡頭光暈」|
| 情誼 → 結尾 | 疊化（Dissolve） | 「轉場」→「基礎」→「疊化」1 秒 |
| 結尾淡出 | 慢速淡黑 | 「轉場」→「基礎」→「淡黑」3 秒 |

---

## 13-5 字幕製作設定

```
方法一：手動添加（推薦，精確對齊）
  步驟 1：點「文字」→「新建文字」
  步驟 2：輸入字幕文字
  步驟 3：字型設定：
    → 字體：思源黑體（剪映內建）
    → 粗細：Bold（加粗）
    → 大小：主引言 32px；次說明 20px；片尾大字 48px
  步驟 4：顏色設定：
    → 字色：白色 (#FFFFFF)
    → 描邊：開啟，顏色深灰 (#333333)，寬度 2px
  步驟 5：位置設定：
    → 主要引言：畫面下方 1/3 處，水平居中
    → 次要說明：右下角
    → 片尾大字：畫面正中央

方法二：AI 字幕（快速，需要人工修正）
  步驟 1：點「文字」→「識別字幕」
  步驟 2：選擇語言「中文（普通話）」
  步驟 3：等待生成後逐一核對修正
  步驟 4：統一套用字幕樣式
```

---

## 13-6 色調調整

```
步驟 1：全選所有素材（長按選第一個 → 點「全選」）
步驟 2：點「調節」→「基礎調節」
步驟 3：套用以下數值：

  亮度（Exposure）：+5
  對比（Contrast）：+10
  飽和度（Saturation）：+8
  色溫（Temperature）：+15（偏暖橙）
  色調（Tint）：+5
  高光（Highlights）：-10
  陰影（Shadows）：+15

步驟 4：點「確認」，套用至全部素材
步驟 5：預覽確認整體色調統一

注意：若某段素材本身已偏暖，
  對該段單獨設定色溫 +8（不要套用 +15）
```

---

## 13-7 最終匯出設定

```
點「匯出」→ 選擇以下設定：

主版（會場播放）：
  解析度：1920×1080
  幀率：30fps
  格式：MP4 H.264
  碼率：高品質

社群版（直式）：
  解析度：1080×1920（9:16 直式）
  幀率：30fps
  格式：MP4 H.264
  碼率：高品質

注意事項：
  ✅ 確認音訊同步（播放前 3 秒和後 3 秒確認對齊）
  ✅ 確認字幕無錯字（逐段快速掃描一遍）
  ✅ 確認結尾淡黑效果正常
  ✅ 確認總片長在 2:58–3:02 之間
```

---

## 13-8 剪映常見問題排解

| 問題 | 原因 | 解決方法 |
|------|------|---------|
| 畫面與音樂不同步 | 素材長度調整後偏移 | 重新對齊音樂軌到影片起始點 |
| 字幕位置跑掉 | 不同素材比例不一致 | 統一所有素材畫面比例 16:9 |
| 匯出影片模糊 | 碼率設定太低 | 改選「高品質」或「極高品質」 |
| 慢速畫面卡頓 | 素材 FPS 不夠 | 改用 60fps 素材做慢速 |
| 合唱聲消失 | 音量鍵幀設錯 | 檢查音量關鍵幀，確認最低不低於 20% |
| 轉場效果消失 | 素材間沒有重疊 | 轉場需要前後素材各有 0.3 秒重疊 |
