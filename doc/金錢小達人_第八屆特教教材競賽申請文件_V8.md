## 壹、設計動機——從特教現場的三重困境出發

### 一、現場觀察：功能性金錢技能的教學斷層

筆者任職特殊教育學校逾十五年，長期擔任生活管理與數學課程教師，教學對象涵蓋輕度至中度智能障礙、自閉症、學習障礙及多重障礙學生。在累積的教學記憶中，有三個場景始終令人無法忘懷，也構成本教材研發最直接的動力。

#### （一）自動化設備前的認知過載

本校一名輕度智能障礙學生，已能在課堂中穩定完成兩位數加減計算，卻在學校走廊自動販賣機前，因為不確定「要先投幣還是先選商品」而反覆嘗試失敗，最終機器吐回零錢、商品未能取得。此後這名學生一遇到自助設備便主動繞路，社區獨立行動的意願因此顯著退縮。

這個案例揭示了特殊教育現場最核心的課程困境之一：**數學計算能力的獲得，並不等同於功能性金錢技能的類化**。找零計算並非純粹的加減法運算，而是涵蓋「辨識幣值→累計投入→比較目標→接收找零→核對驗收」整個交易序列的意義理解。市售教材多停留於靜態紙本或簡報形式，缺乏即時個別化回饋，也無法重現真實設備的操作序列，錯誤認知因而在無結構回饋的練習中持續固化。

#### （二）服務業情境的社交認知挑戰

本校一名輕度自閉症學生，在家長陪同下前往社區理髮廳，由於店家已改用自助點單機（數位看板觸控式），服務項目以分類選單呈現，需先點選「服務類別」才能看見具體項目與價格。這名學生因「類別選單」超出其已知操作序列，在機台前站立近五分鐘無法啟動，家長的語言提示（「你要剪頭髮，按剪髮類別」）因出現在嘈雜環境中，語音混雜背景噪音，未能有效引導。

理髮廳自助點單機的操作，對特殊需求學生而言是一個被嚴重低估的複雜情境——不只是「付錢」，更涵蓋「服務項目辨識」「分類選單導覽」「確認訂單核對」「取號等候」等一系列社交與認知任務。這種服務消費場景在日常生活中極為普遍，卻幾乎未出現在任何現有特教教材的情境設計中。

#### （三）系統性課程工具的長期缺位

《十二年國教身心障礙相關之特殊需求領域課程綱要》生活管理領域，已明確將「財務管理」列為核心學習主題，涵蓋認識貨幣、使用自動化設備（ATM、購票機、自動販賣機）、社區消費技能等子項目。然而課室現實是：教師手邊可用的教材不外乎自製圖卡、靜態簡報，或昂貴的模擬設備。這些工具共同的缺陷是**無法重現設備操作的動態序列**，學生在教室中練習的，永遠是「靜止畫面上的決策」，而非「動態介面中的流程推進」。

本教材的研發，正是為了填補橫跨「數學先備能力 → 貨幣認知建立 → 真實情境類化」三個學習層次、在特教領域長達十年以上的系統性課程工具缺口。

## 貳、設計理念——以科技實踐通用設計學習

### 一、UDL 三大原則的具體落實

本教材以美國 CAST（2018）通用設計學習（Universal Design for Learning）框架為設計基礎，於每一單元均完整落實三大原則。以下以 **A2 理髮廳自助點單機** 單元為具體說明範例。

#### （一）多元表徵（Multiple Means of Representation）

知識呈現不依賴單一感官管道，同一資訊以視覺、聽覺、文字、動覺四種管道同步提供：

- **視覺**：A2 單元服務項目以高解析圖文卡呈現，「一般剪髮」「造型剪髮」「洗吹髮」等各項服務均附有情境插圖與清晰定價；介面仿真真實理髮廳觸控看板佈局，服務分類以顏色區塊編碼，降低選單導覽的認知負擔。九種面額貨幣均提供正反面真實圖片，硬幣 / 紙鈔以色彩差異區分，提供持續性視覺線索。
- **聽覺**：全程中文語音引導（Web Speech API 優先選用 Microsoft 雅婷），金額依台灣傳統唸法轉換（如「一百五十元」而非「一五零元」），特殊處理 11–19 元語音、「二 / 兩」發音差異，確保語音示範符合台灣生活語境；每步操作均有對應語音提示（如「請選擇您需要的服務項目」），形成視覺與聽覺的雙重引導。
- **文字**：操作提示以簡短命令句呈現（5 字以內為原則），避免複雜從屬子句增加語言理解負擔；訂單確認頁以大字體列出服務名稱與金額，並設有「確認」前的語音朗讀機制。
- **動覺**：付款操作以「逐枚點選硬幣 / 紙鈔」方式進行，每次投入均伴隨視覺動畫與音效回饋；取號碼牌的動作設計仿真真實機台（按鈕→列印→取紙），強化動覺記憶鏈結。

#### （二）多元行動與表達（Multiple Means of Action and Expression）

三種操作介面滿足不同動作能力需求：

| 操作模式 | 技術實作 | 適用對象 |
|---------|---------|---------|
| 觸控點擊 | 原生 Touch Events，支援多指觸控防誤觸 | 平板使用者（含行動裝置）|
| 滑鼠點擊 | 標準桌面操作 | 電腦使用者 |
| **輔助點擊模式（Click Mode）** | Action Queue 操作佇列，任意位置點擊執行下一步 | 腦性麻痺、多重障礙、精細動作困難學生 |

**輔助點擊模式**為本教材最具創新性的可及性設計，在現有特教數位教材中屬首創。系統依據當前任務狀態自動建立操作佇列（A2 依序為「選服務類別 → 選服務項目 → 確認訂單 → 付款 → 取號碼牌 → 等候叫號」），任意位置點擊即推進下一步，完全消除精細動作障礙對課程參與的結構性阻礙，直接回應 CRPD 第 9 條無障礙近用的精神。

三種難度設計對應直接教學法的「教師示範 → 引導練習 → 獨立操作」褪除序列：

| 難度 | 教學定位 | 系統行為 | 最適學生特徵 |
|------|---------|---------|------------|
| **簡單模式** | 明示教學（教師示範階段） | 黃色光暈持續標示正確目標；語音逐步引導每一步驟；自動提示下一步 | 重度認知障礙；初次接觸特定情境 |
| **普通模式** | 引導練習（師生共同操作） | 連續三次錯誤後出現提示；語音播題後等待學生操作；逐步驟追蹤錯誤 | 輕中度智能障礙；已具基礎認知但需結構支持 |
| **困難模式** | 獨立練習（類化測試） | 無提示、無引導；仿真真實情境；系統僅記錄結果 | 輕度障礙；ASD；LD；獨立類化練習 |

#### （三）多元參與（Multiple Means of Engagement）

- **個人化意義**：A2 單元教師可預設學生最常使用的服務類別（如某生固定只剪一般剪髮），降低初學階段的選擇焦慮；A1 / A3 / A4 支援自訂商品 / 餐點圖片上傳（魔法商品系統），以學生最喜愛的品項作為練習標的，顯著提升學習動機。
- **可預期性結構**：A2 的服務流程步驟固定且可重複，符合自閉症學生對可預期結構的學習偏好；系統每次操作均從相同介面狀態開始，消除「介面狀態不確定性」的焦慮源。
- **即時正增強**：每步答對後語音讚美，完成全流程後煙火動畫與音效，並連結獎勵積分系統累積點數兌換個人化獎項，與班級代幣制度無縫整合。
- **低壓學習環境**：深色模式減少螢幕藍光刺激，適合視覺敏感或情緒行為障礙學生；數位環境中操作失敗不帶任何社會後果，有效降低焦慮型學生的情境迴避行為。

### 二、工作分析法的系統應用

生活技能的習得，核心在於將複雜操作流程細分為可教、可測的子步驟。本教材 A 系列六個情境單元，均以工作分析法（Gold, 1980）完整拆解真實設備操作流程：

| 單元 | 情境 | 步驟數 | 工作分析核心步驟序列 |
|------|------|--------|-------------------|
| A1 | 自動販賣機 | 5 | 選商品 → 投幣 → 確認金額 → 取貨 → 核對找零 |
| **A2** | **理髮廳自助點單機** | **6** | **選服務類別 → 選服務項目 → 確認訂單 → 付款 → 取號碼牌 → 等候叫號** |
| A3 | 速食店點餐機 | 7 | 瀏覽菜單 → 加入購物車 → 確認訂單 → 選付款方式 → 付款 → 等候取餐號 → 取餐 |
| A4 | 超市購物結帳 | 6 | 選取商品 → 確認價格 → 選付款方式 → 付款 → 核對找零 → 完成 |
| A5 | ATM 操作 | 7 | 插卡 → 輸密碼 → 選功能 → 輸金額 → 取款 / 存款 → 退卡 → 取收據 |
| A6 | 火車票購票機 | 8 | 選出發站 → 選目的地 → 選票種 → 確認票價 → 付款 → 取票 → 核對找零 → 完成 |

**A2 理髮廳自助點單機** 是 A 系列中唯一聚焦於「服務業消費」情境的單元，與其他單元所涵蓋的商品購買場景形成互補。其特殊教育意義體現在以下三點：

- **服務項目辨識**：學生需從多項服務中依個人需求正確選擇，此為服務消費的核心認知任務，與商品購買邏輯（看圖選商品）有本質差異。
- **取號等候的社會性理解**：「取號碼牌 → 坐下等候 → 聽到叫號再起身」是現代服務業的標準流程，也是社區適應的必要社交慣例；A2 單元將此納入工作分析，有助於學生建立「等待」的功能性意義理解。
- **付款金額固定無找零**：A2 單元的付款設計以「確認金額→湊出足額」為主，無找零步驟，降低計算認知負荷，適合用於過渡性訓練（C4 付款技能類化至真實情境）。

### 三、三段式學習歷程的整體規劃

本教材建構「數位模擬 → 紙本形成性評量 → 實地類化」的三段式學習序列：

**第一段：數位模擬練習（課堂主軸）**

學生在軟體中完成情境模擬，教師透過三種難度設定進行差異化教學，即時觀察錯誤類型與規律，作為調整教學策略的依據。

**第二段：紙本形成性評量（課後作業）**

列印對應作業單（18 單元各備 10 種題型可選），評估數位學習是否遷移至紙筆形式，補充個別化書寫練習，形成「數位操作 ↔ 紙本記錄」的雙向學習回路。

**第三段：實地類化應用（社區轉銜）**

以真實理髮廳自助機、超市、ATM 或火車站售票機進行社區操作演練，以行為檢核表記錄各步驟達成程度，作為 IEP 轉銜目標達成評估之依據。

## 參、教材內容概述——三系列 18 單元完整架構

### 一、系統化學習序階

本教材採「先備→認知→應用」三層次螺旋式課程設計，確保每一層次的學習均建立於前一層次的鞏固基礎之上：

**【F 系列・數學基礎先備】（6 單元）**
F1 一對一對應 → F2 唱數 → F3 數字認讀 → F4 數字排序 → F5 量比較 → F6 數的組成

**【C 系列・貨幣認知建立】（6 單元）**
C1 認識錢幣 → C2 數錢 → C3 換錢 → C4 付款 → C5 夠不夠 → C6 找零

**【A 系列・真實情境類化】（6 單元）**
A1 自動販賣機 → A2 理髮廳 → A3 速食店 → A4 超市 → A5 ATM → A6 火車票購票機

## 伍、教學心得與預期效益

### 一、試教觀察：從「他不是不會」到「讓他自己來」

在 A2 初步試教過程中，我觀察到一個令人印象深刻的轉折：一名輕度自閉症學生（乙生），在第四次操作 A2 普通模式時，開始在操作前主動小聲說出「先選剪髮，再選哪一種」——這是他將操作序列語言化的信號，也是程序性記憶即將固化的前兆。

兩週後，這名學生在家長陪同下前往社區理髮廳，面對真實的自助點單機，主動操作完成了服務選擇與取號。家長在聯絡簿上寫道：「他自己按的，我只站在旁邊。機器叫號的時候他自己站起來走過去，完全沒有問我。」

這不只是一個操作技能的類化，更是一次自主性與自信心的建立。特教轉銜教育的核心目標，不是讓學生學會在老師面前示範技能，而是讓他在社區環境中，因為早已練習過，所以知道自己可以。

| 5 分鐘 | 直接教學 | A2 流程總覽 | 教師展示「A2 操作流程步驟海報」（A2 大小，圖文並列）：① 選類別 ② 選服務項目 ③ 確認訂單 ④ 付錢 ⑤ 取號碼牌 ⑥ 坐下等叫號。全班跟讀步驟，教師強調：「步驟③確認訂單：你要先看清楚選了什麼、多少錢，才按確認！」 | 丙生：流程海報僅展示①②⑤（選服務・付錢・取號牌），搭配具體動作示範；乙生：提供個人版步驟小卡（同海報內容，可隨身攜帶） |

## 柒、教學省思與建議

「理髮廳」對一般人而言是再普通不過的地方，每隔幾週就要去一次，從來不需要思考怎麼用那台點單機。但對許多特殊需求學生而言，那台機器代表的是「分類選單的不確定」「等待的焦慮」「出錯後不知道怎麼辦」。

「金錢小達人 A2」的存在，讓學生在第一次走進真實理髮廳之前，已經在安全的數位環境中操作了數十次。他知道「先選類別，再選項目」；他知道「取號牌之後要坐下等」；他知道「叫到我的號碼我才起身」。這些不是被背下來的規則——而是在無數次安全練習中形成的行動記憶。

**給教師的建議**：A2 是 A 系列中「服務業消費」的唯一單元，且操作難度適中（無找零計算），非常適合作為學生從 C4 付款技能進入真實情境類化的第一個 A 系列單元。建議在 C4 連續三次正確後，立即銜接 A2 簡單模式——讓技能遷移趁熱打鐵，不要讓新學的認知在等待中消散。社區實地體驗（第三節）的時機，建議安排在第二節結束後的同週內，越接近類化窗口越好。
