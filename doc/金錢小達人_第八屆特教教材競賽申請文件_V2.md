# 附表三：創作說明

## 壹、設計動機與背景

### 一、現場教學觀察

筆者任教特殊教育學校逾十五年，對象涵蓋輕度至中度智能障礙、自閉症及多重障礙學生。在生活數學與生活管理課程中，金錢技能教學始終面臨三大困境：

**（一）抽象概念難以內化**

對於具體運思期尚未發展完整的特殊需求學生，幣值辨識（尤其是 50 元、500 元面額的混淆）、找零計算、ATM 操作流程等任務，往往需要逐步拆解、反覆練習，方能形成穩定的操作模式。然而，市售輔助教材多停留於紙本或靜態簡報，缺乏即時回饋機制，學生無法在錯誤中立即修正認知。

**（二）安全練習環境匱乏**

功能性金錢技能的習得，最終目的在於社區參與與獨立生活。然而讓學生在真實的超市、便利商店或 ATM 前反覆練習，不僅存在消費糾紛的現實風險，也可能因失敗經驗而加深焦慮、降低學習動機。特殊教育現場迫切需要一個能忠實模擬真實情境、且允許無限次錯誤嘗試的數位練習環境。

**（三）系統性課程工具缺位**

依據「十二年國教身心障礙相關之特殊需求領域課程綱要」，生活管理領域明訂「財務管理」為重要學習主題，涵蓋認識貨幣、使用自動化設備、消費技能等子項目。然而坊間鮮少有系統性、可個別化調整、且適用於中學階段的電腦輔助教材。國中小教師多依賴自製教材，不僅耗費大量備課時間，品質也參差不齊。

### 二、課程法規依據

本教材對應下列課程法規與綱要：

- 《特殊教育法》第 28 條：特殊教育學生應訂定個別化教育計畫（IEP）
- 《十二年國教身心障礙相關之特殊需求領域課程綱要》：生活管理領域——財務管理主題
- 《十二年國民基本教育課程綱要總綱》：生活課程、數學領域、綜合活動領域核心素養
- 《高級中等以下學校特殊教育課程教材教法及評量實施辦法》第 6 條：教材應符合學生個別學習需求

### 三、通用設計學習（UDL）理念

本教材全面落實 UDL 三大原則：

| UDL 原則 | 具體體現 |
|---------|---------|
| **多元表徵（Representation）** | 文字、圖片、語音三軌同步；9 種面額各有正反面高清圖片；語音依台灣傳統唸法轉換 |
| **多元行動與表達（Action & Expression）** | 觸控、滑鼠點擊、輔助點擊模式（任意位置點擊執行下一步）三種操作介面 |
| **多元參與（Engagement）** | 三種難度（簡單/普通/困難）、魔法商品上傳（個人化）、獎勵系統（正增強）、教師可設定題數與出題範圍 |

## 貳、內容概述

### 一、教材架構

「金錢小達人」以三大系列 18 個單元，建構由淺入深的系統性學習歷程：

```
F 系列（數學基礎）──→ C 系列（貨幣認知）──→ A 系列（生活應用）
    6 個單元                  6 個單元                  6 個單元
  F1~F6                     C1~C6                     A1~A6
```

**F 系列——數學基礎**（先備能力培養）

| 單元 | 主題 | 核心能力 |
|------|------|---------|
| F1 | 一對一對應 | 物品與數量配對；拖曳操作 |
| F2 | 唱數 | 1～100 正向唱數；指認數字符號 |
| F3 | 數字認讀 | 阿拉伯數字與中文數字辨識 |
| F4 | 數字排序 | 數字大小比較；拖曳排序 |
| F5 | 量比較 | 多少、大小、長短概念 |
| F6 | 數的組成 | 十進位概念；分解與合成 |

**C 系列——貨幣認知**（幣值概念建立）

| 單元 | 主題 | 核心能力 |
|------|------|---------|
| C1 | 認識錢幣 | 9 種面額辨識；正反面辨認 |
| C2 | 數錢 | 累加計算；拖曳點數 |
| C3 | 換錢 | 等值兌換；幣值等價概念 |
| C4 | 付款 | 選取硬幣/紙鈔付出正確金額 |
| C5 | 夠不夠 | 比較持有金額與商品售價 |
| C6 | 找零 | 計算應找金額；辨識找零組合 |

**A 系列——生活應用**（真實情境模擬）

| 單元 | 主題 | 模擬場景 |
|------|------|---------|
| A1 | 自動販賣機 | 投幣選購、取貨找零 |
| A2 | 理髮廳自助機 | 服務選擇、付款取號 |
| A3 | 速食店點餐機 | 菜單瀏覽、點餐付款取餐 |
| A4 | 超市購物 | 挑選商品、結帳付款找零 |
| A5 | ATM 操作 | 提款、存款、查詢、轉帳 |
| A6 | 火車票購票機 | 路線選擇、票種購票找零 |

### 二、教材特色

**（一）三種難度差異化設計**

| 特性 | 簡單模式 | 普通模式 | 困難模式 |
|------|---------|---------|---------|
| 語音引導 | 全程語音+逐步提示 | 題目語音，錯誤3次出現光暈提示 | 僅語音播題，無提示 |
| 操作輔助 | 黃色光暈標示正確目標 | 部分引導 | 完全自主操作 |
| 適用學生 | 重度認知障礙、初學 | 輕中度障礙、有基礎 | 程度較佳、類化練習 |

**（二）輔助點擊模式（Click Mode）**

針對動作協調困難（如腦性麻痺、多重障礙）學生，啟用後畫面任意位置點擊即執行下一步操作，系統自動建立操作佇列（Action Queue），解決精細動作障礙造成的參與困難。

**（三）魔法商品系統**

A1（販賣機）、A3（速食店）、A4（超市）支援教師/家長上傳自訂商品圖片，賦予課程個人化意義：
- 可上傳學生喜愛的零食、日常採購品
- 配合 IEP 轉銜目標（社區採購）設計個人化購物情境

**（四）作業單系統**

18 個單元各有對應印刷式作業單，共 10 種以上題型：
- 看圖填空（幣值辨識）
- 計算付款金額
- 找零計算
- 圖片連連看
- 「夠不夠買？」判斷題

教師可自行調整題型組合與題數後列印，作為形成性評量工具。

**（五）獎勵系統**

獨立獎勵模組，教師可：
- 上傳學生個人照片
- 設定答對題數轉換點數比例
- 累積點數兌換個人化獎項
- 完整支援 IEP 正增強策略

**（六）全語音中文發音**

使用 Web Speech API 搭配 Microsoft 雅婷（繁體中文）語音引擎，依台灣傳統唸法轉換金額（如「一百二十元」而非「120 元」），特殊規則處理 11-19 元、「二/兩」發音差異，確保語音示範正確性。

## 肆、使用說明與教學建議

### 一、建議教學流程

本教材建議採「數位模擬 → 紙本評量 → 實地類化」三段式學習歷程：

```
【第一階段：數位模擬練習】
  金錢小達人軟體操作
  → 反覆練習、即時語音回饋
  → 教師觀察錯誤類型

【第二階段：紙本形成性評量】
  對應作業單列印
  → 評估保留程度
  → 補充個別化練習

【第三階段：實地類化應用】
  社區購物（超市/便利商店）
  或 ATM 實機操作
  → 配合 IEP 轉銜目標
  → 評估類化成效
```

### 二、融合教育應用

本教材同樣適用於融合班教學情境：

- **差異化作業**：不同障礙程度學生選用不同難度模式，使用相同情境素材，降低同儕標記效應
- **協作學習**：同儕互助模式，一般生扮演「顧客」，特殊需求學生操作軟體扮演「收銀員」角色
- **班級增強系統**：獎勵系統可全班共用，配合班級代幣制度整合

## 一、學生能力描述

本班共 4 名學生，障礙類別與現況如下：

**甲生（輕度智能障礙）**
- 認知：能辨識 1 元、10 元、100 元、1000 元，但混淆 5 元/50 元、200 元/500 元
- 數概念：能計算兩位數加法，進退位偶有錯誤；對抽象找零計算（「你給了100元，東西59元，找回幾元？」）感到困難
- 語言：能以短句表達需求，閱讀國字能力相當國小三年級
- 社會技能：可與熟悉的大人互動，面對陌生店員有焦慮傾向
- 動作：精細動作正常，可操作觸控螢幕

**乙生（自閉症，輕度）**
- 認知：能辨識所有面額，計算能力相當同齡，但不理解「找零」情境意義（知道算法，不知道為何需要找零）
- 語言：口語流暢，但情境適應性低（在真實店面常愣在原地不知如何付款）
- 社會技能：難以理解店員給予模糊指示（如「找你30元」），需要明確視覺提示
- 動作：正常

**丙生（學習障礙，伴隨注意力缺陷）**
- 認知：數學計算能力相當同齡，但閱讀數字有視知覺混淆（6/9、57/75 易混）
- 注意力：持續注意力約 10~15 分鐘，需頻繁切換活動型態
- 語言：能正常溝通
- 動作：正常

**丁生（中度智能障礙）**
- 認知：能辨識 1 元、10 元、100 元；不認識 5 元、50 元、200 元、500 元、2000 元；數概念相當國小一年級
- 語言：能說 3~5 個字的短句，理解簡單指令
- 社會技能：喜歡與人互動，需要大量肢體及語音提示
- 動作：精細動作稍弱，觸控操作需引導

## 二、學習目標

### 課程核心目標

依「十二年國教特殊需求領域生活管理課綱」財務管理主題，本單元目標為：

> 學生能在模擬情境及真實社區環境中，正確辨識台灣流通貨幣，判斷持有金額是否足夠購買指定商品，並完成付款與確認找零之消費流程。

### 各節學習目標

**第一節——認識錢幣（C1、C2）**

| 目標 | 甲生 | 乙生 | 丙生 | 丁生 |
|------|------|------|------|------|
| 指出並說出 5 元/50 元之差異 | ○ | 複習 | 複習 | △（能指出圖片，說出名稱困難）|

## 一、學生能力描述

本組共 3 名學生，均具備本學期 IEP 轉銜目標「能獨立操作 ATM」：

**甲生（輕度智能障礙，15 歲）**
- 認知：能辨識所有幣值，四則運算能力相當國小五年級，能閱讀簡單中文
- 先備能力：曾與家人一起使用 ATM，但不敢自己操作；了解「提款」概念，不了解「存款」「轉帳」
- IEP 長期目標：獨立操作 ATM 完成提款、查詢餘額

**乙生（自閉症，輕度，16 歲）**
- 認知：計算能力佳，但對操作流程的「變化」（如 ATM 畫面跳轉）感到極度不安，需要固定程序
- 先備能力：完全未使用過 ATM；家長反映因「害怕出錯」而抗拒
- IEP 長期目標：能依步驟卡獨立完成 ATM 提款，降低焦慮反應

**丙生（輕度智能障礙，14 歲）**
- 認知：數字辨識正常，計算能力相當國小三年級，能理解簡單圖文指示
- 先備能力：從未使用 ATM；知道 ATM 是「領錢的機器」
- IEP 長期目標：在口語提示下完成 ATM 提款

# 操作手冊摘要（附錄）

## 一、硬體/軟體環境需求

| 項目 | 最低需求 | 建議 |
|------|---------|------|
| 作業系統 | Windows 10、macOS 11、iOS 14、Android 10 以上 | Windows 11 |
| 瀏覽器 | Chrome 90+、Edge 90+ | Chrome 最新版 |
| 螢幕 | 9.7 吋以上 | 13 吋以上（電腦）|
| 語音功能 | 需有繁體中文語音（Windows 內建雅婷、iOS 內建美嘉）| Microsoft 雅婷 |
| 網路 | 首次開啟需連線（載入語音）；其後可離線使用 | 不限 |

## 二、安裝程序

1. 將隨身碟插入電腦
2. 開啟 `index.html`（首頁）或直接開啟 `html/` 中任一單元 HTML
3. 允許瀏覽器使用語音（點擊瀏覽器允許語音通知）
4. 首頁點選目標單元即可開始

**無需安裝任何程式，無需網路（語音首次啟用後）。**

## 三、操作方法

### 設定頁面

各單元均有設定頁，可調整：
- 難度（簡單／普通／困難）
- 題數（5～20 題，依單元而異）
- 輔助點擊模式（啟用後任意位置點擊即執行下一步）
- 音量（依裝置系統設定）

### 獎勵系統

1. 開啟 `reward/index.html` 或點選任一單元「🎁 獎勵」按鈕
2. 新增學生（上傳照片、設定名稱）
3. 各單元完成後點選獎勵按鈕，依答對題數加分
4. 點數達標後兌換預設獎項

### 作業單系統

1. 開啟 `worksheet/index.html` 或點選各單元「作業單」按鈕
2. 選擇單元與題型組合
3. 瀏覽器列印（建議使用「縮放至頁面」選項）

## 四、教學指引

詳見「附表三之一：教學活動設計（一）（二）」。

建議教學順序：F系列（數學基礎先備）→ C系列（貨幣認知）→ A系列（生活應用）。

各單元亦可獨立使用，依學生 IEP 目標選擇適當切入點。
