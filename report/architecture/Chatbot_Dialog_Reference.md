# 金錢小助手 — 對話內容參考表

> 檔案位置：`js/chatbot-engine.js` → `DIALOG` 物件
> 最後更新：2026-04-30

## 說明

| 欄位 | 說明 |
|------|------|
| **節點 ID** | DIALOG 物件的 key，`getRootNode()` 依情境自動選擇入口 |
| **吉祥物說的話** | 顯示在對話泡泡，同時語音朗讀 |
| **選項** | `next` = 跳到另一節點；`reply` = 終端回覆後顯示「再問一個問題」 |

## 情境自動選根節點規則

| 情境條件 | 對應入口節點 |
|---------|------------|
| A1 + phase=selectItem | `a1-selectItem-root` |
| A1 + phase=payment | `a1-payment-root` |
| A1 + phase=change | `a1-change-root` |
| A1（其他） | `a1-root` |
| A2~A6（購物系列） | `a-root` |
| B 系列 | `b-root` |
| C 系列 | `c-root` |
| F 系列 | `f-root` |
| 其他 / 預設 | `global-root` |

> 🪙 硬幣：1元、5元、10元、50元

### `global-money-calc` — 怎麼算要付多少

> 付錢的步驟：
>
> 1️⃣ 先看商品的價格
> 2️⃣ 從大面額的錢開始拿
> 3️⃣ 一張一張加起來
> 4️⃣ 加到「夠了」就可以付！

| 選項 | 動作 |
|------|------|
| 💸 那找零怎麼算？ | → `global-money-change` |
| 👍 我懂了！ | 回覆：很好！先從大面額的開始，就不會選太多 😊 |
| ⬅️ 回主選單 | → `global-root` |

---

### `global-money-change` — 找零算法

> 找零的算法：
>
> 💰 你付的錢 → 50元
> 🏷️ 商品價格 → 35元
>
> 找零 ＝ 50 － 35 ＝ 15元 ✅
>
> 「你付的錢」減掉「商品價格」就是找零！

| 選項 | 動作 |
|------|------|
| ✅ 錢夠不夠怎麼判斷？ | → `global-money-enough` |
| 👍 我懂了！ | 回覆：很好！找零就是「付的」減「價格」，記住這個公式 💪 |
| ⬅️ 回主選單 | → `global-root` |

---

### `global-money-enough` — 夠不夠判斷

> 判斷錢夠不夠：
>
> 你的錢 ≥ 商品價格 → ✅ 夠了！
> 你的錢 ＜ 商品價格 → ❌ 不夠！
>
> 例如：手上 30元，商品 25元 → 夠了！
> 例如：手上 20元，商品 25元 → 不夠！

| 選項 | 動作 |
|------|------|
| 👍 我懂了！ | 回覆：很好！手上的錢大於等於價格就夠了 😊 |
| ⬅️ 回主選單 | → `global-root` |

> → 或直接投一枚 50元也可以！
>
> 投太多沒關係，多的會找回來 😊
