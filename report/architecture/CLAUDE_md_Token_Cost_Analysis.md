## 五、實際情境補充

### 5.1 Claude Code 的特殊行為

Claude Code（本工具）與直接呼叫 API 有以下差異：

| 行為 | 說明 |
|------|------|
| Context 壓縮 | 對話過長時，Claude Code 會自動壓縮**歷史訊息**，但 **CLAUDE.md 不會被壓縮** |
| 每次都載入 | CLAUDE.md 在每個對話 session 開始時全部載入 |
| 費用承擔 | 使用 Claude Code CLI 時，token 費用由 Anthropic 帳號承擔 |

### 5.2 CLAUDE.md 不會被壓縮的意涵

```
歷史訊息（可壓縮） + CLAUDE.md（不壓縮，每輪都全量傳入）
= 隨對話輪數線性增加的 CLAUDE.md token 消耗
```

這表示 CLAUDE.md 越大，**每個 user message 的成本越高**，且不會因壓縮而減少。

### 5.3 現況（91KB）vs 建議目標
