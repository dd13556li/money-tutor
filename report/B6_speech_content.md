## 二、遊戲流程總覽

每關分三個階段：**購物** → **付款** → **找零/結果**

## 三、各模式流程與語音

### 3.1 關卡開場任務說明彈窗（三模式通用）

| 語音 | 文字 |
|------|------|
| 第一關 | `歡迎來到MARKET_NAME！第N關，今天要買：ITEM1、ITEM2、ITEM3，預算X元。` |
| 後續關 | `第N關，今天要買：ITEM1、ITEM2，預算X元。` |

彈窗自動計時 3.2 秒或點擊後關閉 → `afterClose` callback。

---

### 3.2 簡單模式（Easy）— 購物階段

#### 執行步驟

1. **開場彈窗** → 語音播完後關閉
2. **afterClose**（200ms 延遲）→ `_speakMissionItemsOneByOne()`：
   - 每項：`ITEM_NAME，PRICE（中文貨幣）`
   - 全部念完後：`共N樣商品，準備出發！`
3. **攤位切換** 語音：
   - 切換到新攤位且有需要商品：`STALL_NAME，要找ITEM1和ITEM2`
   - 切換到新攤位且已完成：`STALL_NAME，這裡已全部收集！`
4. 點選**清單上的商品**：
   - 語音：`ITEM_NAME，PRICE`（元）
   - 動畫：收集進度 `+1 N/TOTAL`、浮動飛出標籤 `+PRICE元`、清單打勾
5. 點選**不在清單的商品**：
   - 語音：`ITEM_NAME不對，這裡需要NEEDED_ITEM1和NEEDED_ITEM2`（若攤位有需要商品）
   - 或：`ITEM_NAME不在今天的購物清單上`
6. **全部收集完成**：`所有商品收集完成，可以去結帳了！`
7. 點「前往結帳 →」→ 結帳確認清單彈窗

#### 結帳確認清單彈窗
- 語音：`合計X元，預算Y元，確認去付款！`
- 5秒後自動關閉 → 進入付款頁

---

### 3.3 普通模式（Normal）— 購物階段

與簡單模式相同，差異：
- **無**逐項朗讀採購清單（`afterClose` 無動作）
- 其餘攤位切換/商品語音相同

---

### 3.4 困難模式（Hard）— 購物階段

與普通模式相同。

---

## 四、付款階段（簡單/普通/困難）

### 進入付款頁語音（300ms 後）

```
共消費X元，請選擇付款金額。
```

### 面額按鈕操作

- 點面額按鈕 → 累計已付，不播語音（靜默）
- 已付金額達到/超過總額（首次）：
  - 剛好：`金額剛好，可以付款了！`
  - 超過：`超過X元，找零後可以付款！`

### 清除按鈕

```
清除，重新選擇
```

### 提示鈕語音

| 難度 | 提示方式 | 語音 |
|------|---------|------|
| 簡單/普通 | Toast 提示 | `可以用N個X元，M個Y元` |
| 困難 | 彈窗 `_showHardModeHintModal(total)` | `需付X元，可以用N個Y元，M個Z元` |

---

## 五、找零階段

### 5.1 簡單/普通模式（直接顯示結果）

| 情況 | 語音 |
|------|------|
| 不需找零（找零=0） | `精準付款！剛好X元，不需找零，買菜成功！` |
| 需要找零 | `付了X元，找回Y元，買菜成功！` |

### 5.2 困難模式（三選一找零驗算）

#### 出題語音
```
你付了X元，買菜共花了Y元，應該找回多少元？
```

#### 答對語音
```
對了！找回X元，買菜成功！
```
（800ms 後顯示找零結果頁）

#### 答錯語音

| 錯誤次數 | 畫面提示 | 語音 |
|---------|---------|------|
| 第1次 | 顯示算式架構 `付N − 商品M = ?` | `不對喔，找零算太多了/太少了，請再試一次` |
| 第2次+ | 顯示完整算式含答案 | `不對喔，找零算太多了/太少了，請再試一次` |
| 跳過（非 retry 模式） | 高亮正確答案 | `正確答案是找回X元` |

## 六、關卡轉場（每關結束）

非最後一關點「下一關」→ 轉場卡（`_showRoundCompleteCard`）：

```
第N關 完成！
商品清單・共X元｜付Y元｜找零Z元
```
1.5 秒後自動進入下一關（或點擊）。

---

## 七、完成畫面（showResults）

### 第一頁（採購回顧）
- 音效：`success.mp3`（100ms）
- 語音（600ms）：`完成了！來看看採購回顧吧！`

### 第二頁（測驗總結，點「📊 查看測驗總結」）
- 煙火動畫 + 語音（300ms 後）：

| 正確率 | 語音 |
|--------|------|
| 100% | `太厲害了，全部答對了！` |
| ≥80% | `很棒喔，答對了N關！` |
| ≥60% | `不錯喔，繼續加油！` |
| <60% | `要再加油喔，多練習幾次！` |

### 連勝徽章（streak）

| 連勝數 | 語音 |
|--------|------|
| 3連勝 | `三連勝，繼續加油！` |
| 5連勝 | `五連勝，太厲害了！` |

### 表現評價徽章

| 正確率 | 評價 |
|--------|------|
| 100% | 完美 🥇 |
| ≥90% | 優異 🥇 |
| ≥70% | 良好 🥈 |
| ≥50% | 努力 🥉 |
| <50% | 練習 ⭐ |
