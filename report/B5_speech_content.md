## 二、各模式流程與語音

### 2.1 關卡開場介紹卡（三模式通用）

每關開始顯示「第N關」介紹卡，語音播完後自動關閉（或點擊、計時 2.2 秒）。

| 難度 | 語音文字 |
|------|---------|
| 簡單 | `第N關，預算X元，選一些你喜歡的東西吧！` |
| 普通 | `第N關，預算X元，在預算內選購商品。` |
| 困難 | `第N關，預算X元，要精算才不會超支！` |

### 2.2 簡單模式（Easy）

#### 執行步驟

1. **開場介紹卡** → 語音播完後關閉（`dismiss`）
2. **afterClose callback**（200ms 延遲）→ 逐項朗讀必買商品
   - 每項：`必買：NAME，PRICE（中文貨幣）`
   - 全部念完後：`共N項必買，一起挑選吧！`
3. **商品頁** 顯示必買商品（紫色已選，不可取消）+ 選購商品
4. 點選商品（選購）：`NAME，PRICE` + 浮動飛出標籤（`+PRICE元`）
5. 點選已選商品（取消）：`取消NAME`
6. 困難模式翻牌（Easy 無）
7. 點「💡 還能選什麼？」（提示鈕）：
   - 高亮可選商品（2500ms）
   - 若有可選：`還剩X元，可以加選NAME1或NAME2...`
   - 若無可選：`還剩X元，沒有可以加選的商品了`
8. 點「🔊 朗讀已選」：`已選：NAME1 PRICE1元，NAME2 PRICE2元，共Z元`
9. 點「✅ 確認購買！」→ 進入 `_handleConfirm()`

#### 答對語音

| 情況 | 語音 |
|------|------|
| 剛好花完預算 | `完美！剛好花了X元，用完全部預算！` |
| 有剩餘 | `太棒了！共花了X元，還剩Y元！` |

#### 答錯語音

| 錯誤情況 | 語音 |
|---------|------|
| 必買商品未選 | `不對喔，記得要選所有必買的商品喔！` |
| 超出預算 | `不對喔，超出預算了，多了X元，請再試一次！` |

#### 3次錯誤後
自動呼叫 `_showBudgetHint()`（高亮 + 語音提示）

#### 下一關轉場
`第N關`（語音）→ 轉場卡 1.1 秒後關閉 → 進入下一關

---

### 2.3 普通模式（Normal）

與簡單模式相同流程，但：
- **無**逐項朗讀必買商品（`afterClose` 無動作）
- 提示鈕：`_showBudgetHint()`（同簡單模式）
- 3次錯誤後自動提示

---

### 2.4 困難模式（Hard）

與普通模式相同，差異：
- **商品價格隱藏**（選購商品顯示 `??? 元`）
- 首次點選商品 → 翻牌揭示價格，語音：`NAME，PRICE`，**不直接選取**（第二次點擊才選取）
- 提示鈕改呼叫 `_showHardModeHintModal()`：

#### 困難模式提示彈窗語音

```
派對預算X元，必買要Y元，還剩Z元，可以選NAME1PRICE1或NAME2PRICE2
```

若無可選：
```
派對預算X元，必買要Y元，還剩Z元，沒有可加選的商品了
```

## 三、完成畫面（showResults）

### 第一頁：派對回顧
- 音效：`success.mp3`（100ms 後播）
- 語音（600ms 後播）：`完成了！來看看派對回顧吧！`
- 顯示：各關預算使用條形圖、本次採購物品 tags

### 第二頁：測驗總結（點「📊 查看測驗總結」後）
- 煙火動畫（`confetti`）
- 語音（300ms 後）：

| 正確率 | 語音 |
|--------|------|
| 100% | `太厲害了，全部答對了！` |
| ≥80% | `很棒喔，答對了N關！` |
| ≥60% | `不錯喔，繼續加油！` |
| <60% | `要再加油喔，多練習幾次！` |

### 連勝徽章（streak）

| 連勝數 | 語音 |
|--------|------|
| 3連勝 | `三連勝，繼續加油！` |
| 5連勝 | `五連勝，太厲害了！` |

### 表現評價徽章

| 正確率 | 評價 |
|--------|------|
| 100% | 完美 🥇 |
| ≥90% | 優異 🥇 |
| ≥70% | 良好 🥈 |
| ≥50% | 努力 🥉 |
| <50% | 練習 ⭐ |

### 預算效率星評（3顆星）

| 平均使用率 | 顯示 |
|-----------|------|
| ≥90% | ⭐⭐⭐ 預算大師！ |
| ≥60% | ⭐⭐ 不錯的控制！ |
| <60% | ⭐ 繼續練習！ |
